# 命盤板塊：我想怎麼做

命盤（首頁 `/`）的定位是：**「我的生命藍圖」**——以**我**為中心，一眼看到自己的能量與階段，並能快速進到占卜、關係、節奏。

---

## 一、現狀簡述

- **入口**：底部 Tab「命盤」→ 首頁。
- **已有**：標題「天機」、五行能量雷達、填寫出生資料後出現「LIFE BLUEPRINT」靈魂原型卡、「SEASON」卡（工作/感情/自己）、每日簽到、今日智慧、創建靈魂卡 CTA。
- **問題**：  
  - 命盤與「占卜」的差異不夠直觀（都是出生資料 + 結果）。  
  - 「我的」主體感不強，未填資料時首屏沒有「這是我的命盤」的引導。  
  - 八字/四柱沒有在命盤首頁露出，想看重點命理要跑去占卜。  
  - 關係、節奏的入口在 SEASON 卡裡，不夠醒目。

---

## 二、命盤我想做成這樣

### 1. 定位一句話

**命盤 = 我的生命藍圖首頁**  
- 未填資料：強調「建立你的命盤」→ 一鍵填出生資料 → 得到「我的能量 + 我的階段」。  
- 已填資料：首屏即「我的」摘要（五行/主元素/一句話標籤），往下才是詳情與入口。

### 2. 首屏結構（從上到下）

| 區塊 | 內容 |
|------|------|
| **頂部** | 一句話：如「你的命盤」/「我的生命藍圖」，副標可為「基於出生時辰的能量地圖」。 |
| **未填資料** | 大 CTA：「建立我的命盤」→ 打開出生資料彈窗；下方可保留「今日能量」等輕量內容。 |
| **已填資料** | ① **我的命盤摘要**：主元素 + 五行比例條或迷你雷達 + 一句話（如「流動的水脈」）；② **快捷入口**：八字詳解、關係說明書、年度節奏，並標明「占卜」是另一類工具。 |

### 3. 與占卜的區分

- **命盤**：以「我」為中心，展示**我的**能量、我的階段、我的關係/節奏入口；偏「常駐的個人藍圖」。  
- **占卜**：以「工具」為中心，八字/紫微/塔羅等**單次查詢**，可選他人或自己；偏「當下問事、解讀」。  
- 命盤頁可有一行：「想算八字/紫微/塔羅？→ 去占卜」，讓用戶知道「深度解讀」在占卜。

### 4. 內容優先級

- **必做**：  
  - 未填/已填兩種首屏狀態清晰（建立命盤 vs 我的摘要）。  
  - 已填後：靈魂原型卡 + 五行雷達保留，SEASON 卡可改為「當前階段」或「今年重點」並強化「關係」「節奏」入口。  
- **建議做**：  
  - 命盤首頁加「八字摘要」小卡（四柱八字或日主+格局一行），點擊跳轉占卜八字詳解。  
  - 大運/流年若後端有數據，可在命盤頁做「大運/流年」入口，詳情在占卜或獨立頁。  
- **可後做**：  
  - 命盤專屬的「年度節奏卡」內容、關係說明書的預覽一句話。

## 三、具體改動建議（可分批做）

1. **文案**  
   - 命盤標題改為「我的命盤」或「生命藍圖」，副標點出「基於出生時辰」。  
   - SEASON 卡按語種做 i18n，按需改為「當前階段」或「今年重點」。

2. **未填資料時**  
   - 首屏突出一個主按鈕：「建立我的命盤」→ 打開出生資料彈窗。  
   - 其餘內容（雷達、簽到、智慧）可收縮或放在「建立命盤」下方。

3. **已填資料時**  
   - 最上方先「我的命盤摘要」卡（主元素 + 一句話 + 迷你五行或比例）。  
   - 再是現有 LIFE BLUEPRINT + SEASON 卡。  
   - 「看看關係說明書」「年度節奏卡」改為明顯按鈕或卡片，必要時加「去占卜」入口。

4. **八字露出**  
   - 在命盤摘要卡或單獨一小卡顯示「八字摘要」（如日主+格局一行），點擊→ 跳轉 `/oracle/reading?tool=bazi`（或帶當前用戶出生參數），避免用戶不知道八字在哪看。

5. **關係 / 節奏**  
   - 關係：保留「看看關係說明書」並確保跳到關係頁；可加「與某人合盤」等入口。  
   - 節奏：保留「年度節奏卡」並跳到節奏頁；若節奏頁尚未做實，可先占位「即將推出」。

---

## 四、小結

- **命盤** = 我的生命藍圖首頁：建立/查看「我的」能量與階段，並通往八字詳解、關係、節奏。  
- **占卜** = 工具集：八字/紫微/塔羅等單次解讀，可選自己或他人。  
- 優先做：未填/已填兩種首屏、摘要卡、關係與節奏入口明顯化；其次做八字摘要與「去占卜」引導。
