# 符文板块 · 主流对标与待办

对照 Co–Star、Sanctuary、Labyrinthos、测测、今日星座等主流占卜/星座 App 中「符文/神谕/每日签」类功能，整理当前符文小板块的**已做**与**可选待办**。

---

## 一、当前已有（已对标主流）

| 功能 | 说明 |
|------|------|
| **今日符文** | 按日期种子每日一符，同一天同一结果；含符号、名称、正逆位释义与意象解读，拉日活。 |
| **主题选择** | 爱情 / 财运 / 事业 / 健康 / 综合，与主流「选方向再抽」一致。 |
| **问题输入** | 选填问题，用于灵性解读，付费 AI 会结合问题生成。 |
| **多种牌阵** | 单符、三符、五符十字、七符世界，满足简单到进阶需求。 |
| **免费/付费分层** | 免费：基础牌义+位置提示；付费：云端 AI 四层解读（全景镜像 / 阴影瓶颈 / 动力链条 / 转化锚点）。 |
| **解读风格** | 传统 / 平衡 / 灵性，付费解读时生效。 |
| **符文词典** | 24 符释义、正逆位、意象解读，便于学习与查阅。 |
| **保存为图片** | 解读区+牌面一键生成图，便于分享。 |
| **保存到解读记录** | 写入 App 解读历史，可回看。 |
| **复制解读** | 一键复制解读正文到剪贴板，方便粘贴分享。 |
| **再抽一次** | 结果页显式「再抽一次」回到选牌主界面，无需依赖顶部返回。 |
| **内层返回** | 在解读结果页点 App 返回键，先回符文占卜主界面，再点才回星圆。 |
| **免责提示** | 页脚「仅供娱乐与自我觉察，不替代专业建议。请理性看待，你的选择始终自由。」 |
| **前端缓存** | 相同问题+相同抽符+相同风格 7 天内复用本地缓存，减少请求。 |
| **仪式感** | 翻牌动画、音效（翻牌、解读出现时），极光背景。 |

---

## 二、可选待办（按优先级）

### P2 · 体验与合规

| 项目 | 说明 |
|------|------|
| **减少动效** | 尊重系统「减少动态效果」：`prefers-reduced-motion: reduce` 时缩短或关闭翻牌/粒子动画，与 `docs/runes-standalone-测试与优化.md` 一致。 |
| **今日符文分享** | 今日符文区块增加「分享」按钮，生成今日一符的卡片图或文案，便于社交传播。 |

### P3 · 内容与多语言

| 项目 | 说明 |
|------|------|
| **符文图案** | 当前为 Unicode 符号；可逐步增加 SVG/插画符文石，与塔罗/雷诺曼牌面插画策略统一。 |
| **多语言** | 若 App 全局 i18n 已支持，符文页可接收语言参数（如 postMessage），主题/按钮/词典等切换中英繁。 |

### P4 · 运营与留存

| 项目 | 说明 |
|------|------|
| **新手引导** | 首次进入时简短引导：「选主题 → 选张数 → 抽符 → 看解读」或高亮「今日符文」与「符文词典」。 |
| **解读后反馈** | 解读结果页可选「有帮助 / 一般」，用于后续优化提示词与模型（需后端或本地统计）。 |

---

## 三、与主流文档的对应关系

- **日/周/月运、每日一签、星座配对、推送**：见 `主流產品對比與待辦.md`、`占卜产品策划-主流参考方案.md`，属 App 级能力，符文板块已提供「今日符文」与「保存到解读记录」与之衔接。
- **牌面插画**：与 `lenormand-card-image-prompts.md`、雷諾曼/塔羅牌面一致，可后续统一规划资源。
- **音效与无障碍**：与 `runes-standalone-测试与优化.md` 建议一致，当前已有基础音效，可补「减少动效」与静音开关。

---

## 四、总结

符文小板块已具备主流产品中符文/神谕类功能的**核心闭环**：每日一符、多阵型、主题与问题、免费/付费解读、词典、保存与分享（图片+复制+解读记录）、再抽一次与内层返回、免责与缓存。  
后续可按产品节奏补：减少动效、今日符文分享、多语言与插画、新手引导与反馈等，进一步对齐主流体验与合规。
