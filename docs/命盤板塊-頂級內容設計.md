# 命盤板塊 · 頂級內容設計

目標：**別人一看就被吸引住**，比主流軟體更強、且有我們自己的風格。

---

## 一、主流怎麼做（對標）

| 產品 | 特點 | 不足 |
|------|------|------|
| **Co–Star** | 一句話狠、直接、易截圖 | 偏負面、像星座標籤 |
| **The Pattern** | 「稀有」「深度連結」營造專屬感 | 文案偏長、節奏慢 |
| **Sanctuary** | 視覺美、冥想感 | 內容辨識度一般 |
| **星座/八字類** | 個人化、有傳統背書 | 要麼太玄要麼太俗 |

我們要的：**既有「專屬感」和「一句話被擊中」，又要有東方底蘊 + 現代質感，不狠不毒，像一位懂你的智者。**

---

## 二、我們的語調與原則

1. **第二人稱「你」**：像在對一個人說話，不冷冰冰。
2. **一句話能截圖**：每個靈魂原型、每個階段都有一句「可分享的金句」。
3. **東方 + 現代**：用「能量」「命盤」「五行」但不堆術語；用「宇宙」「節奏」但不虛飄。
4. **不討好、不貶低**：不說「你很特別」（廉價），不說「你很難搞」（製造焦慮）；說**看見**——「你的存在感來自行動，不是解釋。」（火）
5. **首屏即鉤子**：未填資料時，第一眼不是「填表」，而是**為什麼要填**——一句讓人想往下看的開場。

---

## 三、首屏內容設計（未填資料）

- **主標（鉤子）**  
  不是「我的命盤」，而是讓人停下來的那一句：  
  **「宇宙在你出生那一刻，已經寫好了你的密碼。」**  
  （可 i18n：英文 "The sky wrote your code the moment you were born."）

- **副標**  
  承諾 + 行動暗示：  
  **「解開它，看見你的能量地圖與靈魂原型。」**  
  （"Unlock it. See your energy map and soul archetype."）

- **主按鈕**  
  不只「建立我的命盤」，可加強行動感：  
  **「解開我的命盤」** 或 **「揭開我的能量地圖」**  
  （"Unlock My Blueprint" / "Reveal My Map"）

- **按鈕下小字**  
  降低門檻、提高期待：  
  **「只需出生日期，約 10 秒。」**  
  （"Just your birth date. About 10 seconds."）

## 四、靈魂原型卡（已填資料）

每個主元素：**標題 + 一句「被看見」的 tagline + 一句可分享金句。**

| 元素 | 標題（現有可保留） | Tagline（讓人被擊中） | 金句（可單獨展示/分享） |
|------|---------------------|------------------------|---------------------------|
| **火** | 燃燒的火光 | 你的存在感來自行動，不是解釋。 | 你不是在燃燒，就是在點亮別人。 |
| **水** | 流動的水脈 | 你不需要被所有人理解，只需要被對的人讀懂。 | 深流不爭，卻能穿石。 |
| **木** | 成長的樹靈 | 你的節奏是生長，不是衝刺。 | 向下紮根，向上發光。 |
| **金** | 鋒銳的金意 | 你的清晰不是冷漠，是尊重彼此的邊界。 | 精煉過的火，才能照見真相。 |
| **土** | 承載的大地 | 你的穩不是慢，是讓別人敢在你面前落地。 | 承載萬物的人，不必證明自己。 |

**卡片結構建議**：  
標題（如「燃燒的火光」）→ 換行 → **Tagline**（加粗或高亮）→ 金句（小一號、可引用樣式）。

## 五、當前階段卡（今年/本階段）

- **區塊標題**  
  **「這一程」** 或 **「此時此刻」**（"This Chapter" / "Right Now"）

- **階段主題名**（可依年/大運替換，目前先用一句通用）  
  **「定錨與發芽」**（"Anchoring & Sprouting"）  
  副句：**「宇宙在推你整理節奏、打好地基——比立刻看到結果更重要。」**

- **工作 / 感情 / 自己**  
  各一句話，利落、可記：
  - **工作**：優先補齊基礎，少在陌生賽道冒險。
  - **感情**：看清相處的節奏，比急著要一個「名分」更關鍵。
  - **自己**：建立規律與邊界，讓能量回到自己身上。

---

## 六、去占卜 / 關係 / 節奏入口

- **去占卜**  
  主按鈕：**「八字 · 紫微 · 塔羅」** 或 **「去占卜」**  
  副文：**「想看完整八字、流年大運、塔羅牌陣？在這裡。」**

- **關係說明書**  
  **「誰與你同頻？誰與你互補？」** → 點擊進關係。

- **年度節奏卡**  
  **「這一年的節奏與時機」** → 點擊進節奏。

---

## 七、與主流差異小結

| 維度 | 主流常見 | 我們 |
|------|----------|------|
| 首屏 | 直接要資料或星座標籤 | 先給「為什麼」：宇宙寫好了你的密碼 |
| 原型 | 形容詞堆砌（熱情、敏感） | 一句「被看見」的 tagline + 一句金句 |
| 語氣 | 要麼毒舌要麼討好 | 看見、尊重、不評判 |
| 文化 | 純西洋或純傳統 | 東方能量 + 現代表達，可中英雙語 |
| 分享欲 | 靠「稀有」標籤 | 靠「說中我」的金句，自然想截圖 |

## 九、細節與「眼前一亮」（已落地）

為讓呈現更豐富、更有辨識度，已做以下增強：

| 項目 | 說明 |
|------|------|
| **首屏裝飾** | Hero 下方三點裝飾線（· · ·），輕微進場動效 |
| **未填收尾句** | CTA 按鈕下增加「解開後，專屬於你的靈魂原型與金句將揭曉」 |
| **元素視覺** | 靈魂原型卡依 火/水/木/金/土 使用不同左側強調色與背景漸層，一眼辨識 |
| **靈魂原型標** | 卡片標題改為「你的靈魂原型」+ 元素符號（🔥💧🌿◇◉） |
| **能量關鍵詞** | 每元素一句關鍵詞（如 行動力·直覺·感染力），資訊更飽滿 |
| **金句贈語** | 金句下方增加「這句送給今天的你。」，強化禮物感 |
| **今日宇宙提示** | 「這一程」卡頂部增加「今日宇宙提示」，依星期輪播 7 則短句，每天不同 |
| **進場動效** | 靈魂原型卡、這一程卡使用 framer-motion 輕微上浮進場，層次更清晰 |
