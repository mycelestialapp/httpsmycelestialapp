# 與主流產品對比：我們已做與待辦

本項目對標玄學/占卜/星座類 App（如 Co–Star、The Pattern、測測、準了等）及通用帳號/設置習慣，整理「已做」與「建議待辦」，方便產品與開發對齊。

---

## 一、帳號與個人資料（已做 ✓ / 待辦 ○）

| 項目 | 狀態 | 說明 |
|------|------|------|
| 註冊時填寫姓名 | ✓ | 註冊表單有「姓名 / 暱稱」，並寫入 Supabase + 同步到 profiles |
| 登入後看到自己的名字 | ✓ | 賬戶中心顯示 display_name，可從 profiles 或 user_metadata 取得 |

## 五、建議優先待辦（決策參考）

1. **頭像後台**：按 `SUPABASE_AVATAR_SETUP.md` 在 Supabase 加上 `avatar_url` 與 `avatars` 儲存桶，頭像上傳與頂欄頭像即完整可用。
2. **生日編輯**：在賬戶中心或設置中增加「生日」顯示與編輯，寫回 `profiles.birthday`。
3. **隱私與條款**：在設置中補上隱私政策與用戶條款連結或內嵌頁。
4. **通知**：若需推送/站內通知，再補後端與前端開關。

其餘可依產品節奏逐步補齊；頭像與修改密碼已按主流習慣做成「可顯示、可修改、站內完成」。

## 六、占卜 / 星圖 / 日活（與主流對比 · 待改進）

對標：Co–Star、The Pattern、Sanctuary、測測、今日星座、CHANI 等。我們已有：星圖首頁、靈魂原型、八字/占星/塔羅/神諭卡/雷諾曼（含大桌、牌義庫）、關係頁、節奏頁、解讀記錄、訂閱入口。

### 6.1 主流標配我們還缺的

| 維度 | 主流做法 | 我們現狀 | 建議 |
|------|----------|----------|------|
| **日/周/月運勢** | 日運一句話、周運、月運是標配，拉日活 | 僅有節奏頁年度主題、cosmicTip 類今日一句，無結構化日/周/月 Tab | 在節奏頁做「日 / 周 / 月」Tab；星圖頂部強化「今日一句」並可鏈到完整日運 |
| **每日一簽/每日一牌** | 每日一牌、每日一簽，固定理由打開 App | 有每日智慧/金句，未做成「每日一牌」儀式（同一天同一張） | 首頁或獨立入口：按日期種子每日一牌 + 2～3 句解讀，可與日運合併 |
| **星座配對 / 合盤** | 選兩星座或兩人生日 → 匹配度 + 一句話，易傳播 | 關係頁有檔案、命盤；無 12×12 星座配對、無輕量雙人合盤 | 關係頁加「星座配對」與「合盤」入口；先做 12×12 配對矩陣，再做雙人生日合盤一句話 |
| **推送** | 每日一條今日星象/今日簽，拉回用戶 | 設置裡通知為佔位 | 日運穩定後接推送（需後端/推送服務），文案偏娛樂/自我覺察 |
| **分享圖 / 邀請** | 結果頁生成圖、分享給好友、帶 ref/soul 拉新 | 有 ref/soul 參數、部分分享；未系統化「分享圖」與「分享給 ta」 | 靈魂原型/今日簽/配對結果可生成分享圖；配對結果頁強「分享給 ta」按鈕 |
| **雷諾曼/塔羅牌面圖** | 牌面為插畫圖 | 目前多為符號/文字，無統一牌面插畫 | 可逐步替換為插畫資源（見 `docs/lenormand-card-image-prompts.md` 等） |

### 6.2 內容與體驗可加強的

| 維度 | 說明 |
|------|------|
| **節奏頁** | 目前僅年度時間軸 + 一組通用主題；可加「日 / 周 / 月」結構，與日運體系打通。 |
| **能量地圖/五行** | 若用戶覺得「平衡分析」有被評判感，可改為「五行氣質」表述，或設為可隱藏（見策劃四）。 |
| **雷諾曼付費** | 大桌解讀的付費鎖（陰影/禪意等）已暫時關閉，後續可接真實支付與權限。 |
| **符文/數字命理/靈擺** | 星圖入口已有，若尚未實現完整抽籤/解讀，可標註「即將推出」或補齊。 |

### 6.3 建議優先級（落地順序）

| 優先級 | 項目 | 理由 |
|--------|------|------|
| **P1** | 節奏頁日/周/月運勢結構 + 星圖「今日一句」強化 | 主流標配，已有 cosmicTip/季節提示可復用 |
| **P1** | 星座配對（12×12）+ 關係頁入口 | 傳播強、關係頁已有基礎 |
| **P2** | 每日一簽/每日一牌（日期種子 + 短解讀） | 提升日活與打開習慣 |
| **P2** | 雙人合盤輕量版（兩人生日 → 一句話） | 與配對形成「關係」閉環 |
| **P3** | 推送、分享圖、邀請鏈接強化 | 依運營與變現節奏 |
| **P3** | 牌面插畫、能量地圖語義優化 | 體驗與差異化，可迭代 |
