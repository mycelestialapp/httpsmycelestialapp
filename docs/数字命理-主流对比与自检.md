## 三、我们的优势（可强化卖点）

1. **解读质量与结构**  
   - 五数齐全（生命路径、命运、灵魂驱动、个性、成熟），且单数解读维度多：core / challenge / career / love / advice / lifeStage / innerChallenge / outerOpportunity / relationshipHint / spiritualGuidance / oracle 等，不少主流 App 只做到基础几项。

2. **神谕版（oracle）**  
   - 有完整「神谕体」长文案，适合做首屏金句、分享语、付费或品牌差异化；主流产品多为短句，我们可主打「深度与意境」。

3. **大师数 11/22/33**  
   - 明确识别、不强制归约，并有扩展解读字段（如 lifeStage、spiritualGuidance），对专业用户友好。

4. **流年透明**  
   - 个人年公式与化简步骤可见（`getPersonalYearSteps`），教育意义强，适合「想弄懂怎么算」的用户。

5. **与命盘档案、关系页打通**  
   - 结果可保存到命盘档案（含当时解读快照），在关系说明书页按档案查看「数字命理解读」并跳转完整结果，形成「一人一档、多工具合一」的动线。

6. **分享图与图书馆**  
   - 支持生成分享图；数字图书馆 + 单数详情页结构清晰，1–9、11、22、33 全覆盖。

7. **技术实现**  
   - 毕达哥拉斯体系实现清晰、可扩展（如加算法、加数字类型），前端与 archives/readingHistory 已接好，后续加功能成本可控。

## 四、建议优先级（产品向）

| 优先级 | 方向 | 说明 |
|--------|------|------|
| P0 | 中文姓名 | 拼音首字母或笔画数映射，或明确标注当前算法含义，避免误解 |
| P1 | 三角图/九宫格 | 至少一种可视化（三角图或九宫格），提升「专业感」与留存 |
| P1 | 两人配对 | 生命路径（或五数）配对解读，与关系说明书、命盘档案联动 |
| P2 | 每日数字 | 今日幸运数/色/一句话，与今日宜忌并列 |
| P2 | 阅读历史 | 确认数字命理写入阅读历史并在统一入口展示 |
| P3 | 个人月/日、Pinnacle | 按用户需求与内容储备再开 |
