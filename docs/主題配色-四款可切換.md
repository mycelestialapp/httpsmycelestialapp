# 主題配色 · 六款可切換

以當前星盤介面（深海藍 + 星空 + 暖金）為基準，共 **6 款** 主題供使用者切換。三款新增主題與預設、以及彼此之間 **顏色差距明顯**，且皆 **貼合實際使用**（對比度、可讀性、長時間閱讀舒適）。

---

## 一、主題一：星際（預設）

- **名稱**：星際 / 深海藍金  
- **背景**：深海藍、星空、星點與幾何光點  
- **主色**：暖金（按鈕、標題、高亮、邊框）  
- **感受**：宇宙、星盤、高級感  
- **對應**：現有 `:root`，不需改動  

---

## 二、主題二：森林

- **名稱**：森林 / 墨綠琥珀  
- **背景**：深墨綠、偏青綠的夜空感（可保留星點，改為偏綠白）  
- **主色**：琥珀 / 古銅（取代金）  
- **感受**：自然、塔羅、神秘森林、大地  
- **差異**：整體偏 **綠**，與藍、紫、灰都明顯不同  

| 變數 | HSL 建議 |
|------|----------|
| --background | 160 35% 5% |
| --card | 160 28% 8% |
| --gold | 35 85% 55% （琥珀） |
| --accent | 55 55% 52% |
| --border | 160 20% 22% |
| --muted | 160 18% 12% |

---

## 三、主題三：皇室

- **名稱**：皇室 / 深紫銀

- **主色**：銀 / 鉑金感（取代暖金）  
- **感受**：神秘、皇室、占星、高冷  
- **差異**：整體偏 **紫**，主色為 **銀**，與藍金、綠琥珀、暖炭都不同
