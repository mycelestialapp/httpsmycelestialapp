# 字體大小與顏色 · 對照主流占卜/占星 App

## 主流產品的常見問題（調研摘要）

- **Co-Star**：選單與區塊標籤字體偏小、副標「rather small」、淺灰文字對比不足，影響可讀性。
- **塔羅/占卜類 App**：業內普遍痛點是「字體偏小 + 文字與背景對比不足」；較好產品會強調**對比清晰、間距足夠**，避免過多淺灰。

## 本專案已做的對齊調整

### 1. 全局顏色對比（`src/index.css`）

- **`--text-subtitle`**：由 `0 0% 84%` 調整為 `0 0% 90%`，副標/說明文字更易讀。
- **`--muted-foreground`**：由 `58%` 調整為 `68%`，次要文字在深色底上對比更足。
- 以上變量在多主題下已一併更新，保持一致性。

### 2. 字體大小（對齊「正文 ≥16px、可讀副標 ≥14px」）

- **聖殿入口（預設）**：標籤與提示由 11px → 14px；副標由 text-xs → text-sm；輸入框由 text-sm → text-base；「從參考問題選」按鈕與分類標籤 11px → 14px；問題列表 text-xs → text-sm；解讀記錄按鈕 11px → 14px。能量球「FOCUS」由 10px → 13px，並改為實色金。
- **塔羅入口**：此前已設為 15–18px 區間，維持不變。
- **解讀結果（MasterOracleReadingView）**：鏡像標籤 15→16px、內文 17→18px；靈魂核心內文 17→18px；副標改用 `text-subtitle` 且 15px，不再使用透明度壓低對比。
- **占卜頁標題**：工具描述統一為 `text-sm text-subtitle`，不再使用 text-xs / muted。

### 3. 顏色使用原則

- 重要說明、按鈕文字：避免單獨使用 `text-muted-foreground`，改用 `text-subtitle` 或 `text-foreground`，以符合主流「高對比」。
- 佔位符：由 `placeholder:text-muted-foreground/60` 改為 `placeholder:text-foreground/55`，在深色底上更易辨認。
- 金色強調：標籤與按鈕上的金色由 `text-gold-strong/90` 改為 `text-gold-strong`，減少發灰感。

## 後續可選

- 若仍需加強可讀性：可將正文統一為 17–18px（接近 iOS 建議的正文大小）。
- 若支援「動態字體」：可跟隨系統字體大小設定（如 Co-Star）做縮放。
