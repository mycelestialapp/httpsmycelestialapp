# 西方占卜 · 層級示意優化說明

在「內容滿意」的基礎上，對第一～第三層的**細節**做了優化，方便落實現稿時有據可依。

---

## 一、第一層（星盤入口）

| 優化項 | 說明 |
|--------|------|
| **背景** | 深色 #0a0e17 → #12182a 微漸層，統一景深 |
| **標題區** | 「星盤」大標 + 副標「西方 · 本命與問事」+ 一條細金線分隔，層次清晰 |
| **區塊標** | 上方加「占卜方式」小標（11px、大寫、字距），便於掃描 |
| **列表行** | 每行約 56px 高、12px 行距；左圖示區 40px、右箭頭，觸控區夠大 |
| **卡片** | 1px 金邊（約 20% 透明度）、16px 圓角、12px 內邊距，視覺統一 |
| **底部導航** | 當前 tab 用金色高亮，與整體風格一致 |

## 二、第二層（單一工具選項，以塔羅為例）

| 優化項 | 說明 |
|--------|------|
| **頁標** | 「塔羅」+ 副標「選擇占卜方式」，標題下留 24px 呼吸 |
| **選項卡** | 三個大卡、16px 間距；卡內 16px 標題（金）+ 13px 說明（灰） |
| **問題類型** | 獨立一區：「問題類型」小標 + 四個 pill（工作 / 感情 / 自我 / 抉擇），選中為金色填充 |
| **主按鈕** | 底部全寬「開始占卜」，金邊漸層 + 明確 CTA |
| **柵格** | 整體以 8px 為基準，對齊與間距一致 |

## 三、第三層（解讀結果）

| 優化項 | 說明 |
|--------|------|
| **頁標** | 「解讀結果」，不搶戲 |
| **牌面** | 單張牌有黃金比例框（約 2:3）、金邊，留白適中 |

| **小提示** | 可加「這張牌給你的提醒」或一句總結，11px 弱化 |
| **按鈕** | 「再抽一張」（描邊金）與「分享」（實心金）並排、等寬、12px 間距 |
| **免責** | 底部小字「僅供娛樂與自我反思」，合規且不搶眼 |

---

## 四、通用細節

- **字體**：標題用 serif（如 Playfair / Noto Serif），內文用 sans，與現有 app 一致。
- **色彩**：主色金 #FFD700 或 hsl(47 90% 70%)，背景深藍，對比與可讀性兼顧。
- **觸控**：主要按鈕與列表行至少 44px 高，方便拇指操作。
- **一致性**：三層都保留頂部返回、標題層級、金/灰層次，便於擴到其他工具（神諭卡、符文等）。
