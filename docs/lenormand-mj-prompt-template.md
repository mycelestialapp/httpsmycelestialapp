# 雷諾曼 / 黑金燙金 · MJ 牌面 Prompt 模板

適用於「神聖幾何 + 燙金 + 黑曜石底板」風格的牌面圖。把 `[CARD_NAME]` 換成具體牌名（英文，如 Rider、Clover、Ship 等）後複製到 Midjourney。

---

## 母版 Prompt（每張牌通用）

```
A mystical holographic sigil of [CARD_NAME], floating sacred geometry, intricate liquid gold filigree on a matte obsidian slab, ethereal bioluminescence, hyper-detailed lines, cinematic shadows, luxury tarot aesthetic, black and gold theme --ar 2:3 --v 6.0 --stylize 750
```

---

## 使用方式

1. 將 `[CARD_NAME]` 替換為該張牌英文名（例：`Rider`、`Clover`、`Ship`、`House`、`Tree`…）。
2. 複製整段到 Midjourney 生成。
3. 下載後可存至 `public/lenormand/` 或 `public/oracle/custom/`，檔名建議與牌 id 對應（如 `rider.jpg`、`clover.jpg`）。
